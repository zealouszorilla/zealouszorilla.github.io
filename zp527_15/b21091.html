Features and Specs

The 7" LCD display is an IPS LCD display which connects to the RPI through the HDMI FPC cable.

The Physical resolution of LCD display is 1024*600.

It has 5 point capacitive touchscreen and connects to RPI1,2,3 via PI3 USB board or to RPI4 via PI4 USB board..

It has 2x3W speakers.

It has a rs232/rs485 interface.

It has a minipcie socket for SIM7600 4G module or SIM868 2G Module.

It has a NGFF 2242 SSD USB3.0 module for Pi 4 installation location

It is compatible with RPI1 model A+ B+/ PI 2 model B/PI 3 model A+ B B+/PI 4 model B nd 2B.

It is powered by the USB TypeC socket and the RPI is powered by the display. Connect RPI1,2,3 with micro usb board, and connect RPI4 with USB Typec board..

It doesn"t need install driver.
