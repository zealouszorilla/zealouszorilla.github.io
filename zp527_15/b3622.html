なだらかなカーブを描くウッドベースに優しい光を灯すファブリックセードを合わせたテーブルライトです。レトロな佇まいがゆっくりと流れる静かな時間を演出します。本体部分は木を贅沢に、緩急をつけたフォルムに削り出しました。なめらかな肌触りとリッチな木の質感が落ち着いた空間を作り出します。本体から下がるボールチェーンのプルスイッチがレトロな表情を盛り上げます。スイッチの先端にはウッドボールを配しました。生成り色のファブリックセードは上下が開いている円柱状なので、上下にも光が漏れる構造です。ファブリックセードを通して優しい光が広がります。コンパクトなサイズ感なのでベッドサイドやシェルフの上など、様々な場面でお使いいただけます。
 
 
- Features -
Function 01
削り出し

木を贅沢に、緩急をつけたフォルムに削り出しました。なめらかな肌触りとリッチな木の質感が落ち着いた空間を作り出します。
   
Function 02
スイッチ

本体から下がるボールチェーンのプルスイッチがレトロな表情を盛り上げます。スイッチの先端にはウッドボールを配しました。
   
Function 03
ファブリック

生成り色のファブリックセードは上下が開いている円柱状なので、上下にも光が漏れる構造です。セードを通して優しい光が広がります。
   
Function 04
コンパクト

コンパクトなサイズ感なのでベッドサイドやシェルフの上など、様々な場面でお使いいただけます。
 
Function 05
サイズ感

デザインを損なわず実用性を両立させた、ベストなサイズ感を追求。

「人が過ごす場所と時間を心地よく」

インターフォルムという社名には、「世界に通用するデザインを」という意味が込められています。
そんな理想を持ち、世界のトレンドと、日本の緻密なデザインワークをかけ合わせた商品づくりを続けています。
